# spring-webflux-practice
spring-webfluxに置き換え

# Mono
単体のオブジェクト(object)

## Mono.just()　
単体のMonoオブジェクトを返す

## Mono.zip()　
複数のMonoオブジェクトを返す

# Flux
複数のオブジェクト(List)

## Flux.just()　
単体のFluxオブジェクトを返す

## Flux.zip()　
複数のFluxオブジェクトを返す
